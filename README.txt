Данный мини-проект создан в качестве тестового задания для компании Effective Mobile.
В проекте представлена основная часть приложения, написанного на FastAPI, и тестовая часть.
Основное приложение находится в каталоге src и состоит из двух приложений orders и products.
Для создания миграций бд  использовалась библиотека uvicorn.
Тестовая часть сделана не до конца, можно протестировать отдельные тесты в ручную.
