Данный мини-проект создан в качестве тестового задания для компании Effective Mobile.
В проекте представлено веб-приложение, написанного на FastAPI, и тестовая часть.
Основное приложение находится в каталоге src и состоит из двух частей orders и products.
Для создания миграций бд  использовалась библиотека uvicorn.
Для тестирования необходимо создать с помощью миграций тестовую базу данных. Тесты написаны с
помощью pytest, pytest-dotenv.
Настройки к основной и тестовой бд находятся в (.env и .test.env).
